libossia
===

A library to expose the parameters of your creative coding application
over the network, and score them in time.

Protocols :
* OSC
* MIDI
* Minuit
* OSCQuery (in progress)

Bindings (in progress) :
* PureData
* C
* Python
* Java (preliminary)
* Unity3D
* Qt and QML

Documentation :
